# Flutter Painter App

A fully-featured, smooth drawing and painting application built with Flutter. It provides an intuitive canvas where users can create digital art with customizable brushes and colors.

## Features

- **Freehand Drawing:** Smooth, responsive freehand drawing directly on the screen using Flutter's `CustomPainter`.
- **Customizable Brush:** An easy-to-use slider allows precise control over the stroke width (from 1.0 to 30.0 pixels).
- **Color Palette:** A curated palette of vibrant colors including black, white (for erasing), and various primary and secondary colors.
- **Undo and Redo:** Built-in history lets you undo mistakes and redo them, providing full control over the creation process.
- **Clear Canvas:** Easily erase the entire canvas with a single tap to start over.

## Tech Stack

- **Framework:** Flutter / Dart
- **UI Architecture:** Stateful widgets for state management, `CustomPainter` for high-performance drawing.

## Getting Started

1. Ensure you have the [Flutter SDK](https://flutter.dev/docs/get-started/install) installed.
2. Clone this repository.
3. Run `flutter pub get` to fetch dependencies.
4. Run `flutter run` to launch the app on your preferred connected device or emulator.
